Gui::ContainerDialog Class Reference

#include <Gui/WidgetFactory.h>

Detailed Description

The ContainerDialog class acts as a container to embed any kinds of widgets that do not inherit from QDialog. This class also provides an "Ok" and a "Cancel" button. At most this class is used to embed widgets which are created from .ui files.
