Gui::StdWorkbench Class Reference

#include <Mod/Web/Gui/Workbench.h>

Detailed Description

The StdWorkbench class defines the standard menus, toolbars, commandbars etc. To define own workbenches you should inherit from StdWorkbench instead of Workbench to have defined the standard GUI elements.
