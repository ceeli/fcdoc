#include <Gui/InputField.h>

Detailed Description

The InputField class The input field widget handles all around user input of Quantities. That includes parsing and checking input. Providing a context menu for common operations and managing default and history values. Although it's derived from a QLineEdit widget, it supports most of the properties and signals of a spin box.

◆ setUnit()

void Gui::InputField::setUnit ( const Base::Unit &  )

sets the Unit this field is working with. After setting the Unit the field will only accept user input with this unit type. Or if the user input a value without unit, this one will be added to the resulting Quantity.
