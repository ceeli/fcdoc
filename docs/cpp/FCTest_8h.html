◆ ADD_OBJECT

#define ADD_OBJECT (   t)    addObject<t>(#t)

Calls DocumentTest::addObject with matching arguments.

FreeCADs typesystem is based on strings while the native C++ typesystem uses template arguments. Those two can't communicate which forces DocumentTest::addObject to require the typename as a string (for FreeCAD) and as a template argument (for casting). This macro only requires the type argument once and calls DocumentTest::addObject with that argument used twice.

This macro should be used in most cases instead of calling DocumentTest::addObject directly because it's less error prone. For info about the return value and exceptions see DocumentTest::addObject.
