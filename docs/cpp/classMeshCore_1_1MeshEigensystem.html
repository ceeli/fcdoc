MeshCore::MeshEigensystem Class Reference

#include <Mod/Mesh/App/Core/Evaluation.h>

Detailed Description

The MeshEigensystem class actually does not try to check for or fix errors but it provides methods to calculate the mesh's local coordinate system with the center of gravity as origin. The local coordinate system is computed this way that u has minimum and w has maximum expansion. The local coordinate system is right-handed.
