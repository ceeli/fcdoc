MeshCore::MeshRefEdgeToFacets Class Reference

#include <Mod/Mesh/App/Core/Algorithm.h>

Detailed Description

The MeshRefEdgeToFacets builds up a structure to have access to all facets of an edge. On a manifold mesh an edge has one or two facets associated.

Note
If the underlying mesh kernel gets changed this structure becomes invalid and must be rebuilt.
