Gui::DocumentObserverPython Class Reference

#include <Gui/DocumentObserverPython.h>

Detailed Description

The DocumentObserverPython class is used to notify registered Python instances whenever something happens to a document, like creation, destruction, adding or removing viewproviders or when viewprovider property changes. This is the equivalent to the app python document observer
