Gui::ActiveObjectList Class Reference

#include <Gui/ActiveObjectList.h>

Detailed Description

List of active or special objects This class holds a list of objects with a special name. Its mainly used to points to something like the active Body or Part in a edit session. The class is used the viewer (editor) of a document.
