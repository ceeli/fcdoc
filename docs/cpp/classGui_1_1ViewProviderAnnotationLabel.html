Gui::ViewProviderAnnotationLabel Class Reference

#include <Mod/TechDraw/Gui/ViewProviderAnnotation.h>

Detailed Description

This is a different implementation of an annotation object which uses an SoImage node instead of an SoText2 or SoAsciiText node. This approach gives a bit more flexibility since it can render arbitrary annotations.
