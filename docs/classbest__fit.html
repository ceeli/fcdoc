best_fit Class Reference

#include <Mod/Cam/App/best_fit.h>

Detailed Description

The main class for the best_fit routine.

It takes a mesh and a Topo_Shape as it's input parameter.

As output, it gives a transformed mesh (rotation + translation) based on a weighted ICP-Algorithm (ICP: Iterative Closed Point) which fits the Topo_Shape
