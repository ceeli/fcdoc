MeshCore::MeshEvalInvalids Class Reference

#include <Mod/Mesh/App/Core/Degeneration.h>

Detailed Description

The MeshEvalInvalids class searches for as 'Invalid' marked facets and points. Basically this comes from a not properly implemented algorithm that marks facets or points as 'Invalid' without removing them from the mesh kernel.
