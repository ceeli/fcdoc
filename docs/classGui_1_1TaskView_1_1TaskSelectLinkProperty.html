#include <Gui/TaskView/TaskSelectLinkProperty.h>

Detailed Description

General Link/Selection editor for the Task view This can be used as part of a TaskDialog to alter the content of a LinkProperty by user input/selection. If set active it reflects the selection to the Property given and acts due the selection filter given to the constructor. It will allow only allowed elements to be selected (SelectionFilter) and shows by the background color if the selection criterion is met. With the call of accept() or reject() the result gets permanent or discarded in the given Property.
