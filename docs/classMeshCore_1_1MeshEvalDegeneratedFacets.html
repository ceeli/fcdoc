MeshCore::MeshEvalDegeneratedFacets Class Reference

#include <Mod/Mesh/App/Core/Degeneration.h>

Detailed Description

The MeshEvalDegeneratedFacets class searches for degenerated facets. A facet is degenerated either if its points are collinear, i.e. they lie on a line or two points are coincident. In the latter case these points are duplicated. If a facet refers to at least two equal point indices then the facet is also regarded is 'corrupt'.
