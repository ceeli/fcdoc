◆ completeExtension()

std::string Base::FileInfo::completeExtension ( ) const

Returns the complete extension of the file. The complete extension consists of all characters in the file after (but not including) the first '.' character.

FileInfo fi( "/tmp/archive.tar.gz" );
ext = fi.completeExtension(); // ext = "tar.gz"

◆ extension()

std::string Base::FileInfo::extension ( ) const

Returns the extension of the file. The extension consists of all characters in the file after (but not including) the last '.' character.

FileInfo fi( "/tmp/archive.tar.gz" );
ext = fi.extension(); // ext = "gz"
