App::Range Class Reference

#include <App/Range.h>

Detailed Description

The Range class is a spreadsheet range iterator. It takes a starting (row, col) and an ending (row, col). Notice that ranges are always at least one element. The next() functions is therefore used e.g as follows:

do { ... while (range.next());
