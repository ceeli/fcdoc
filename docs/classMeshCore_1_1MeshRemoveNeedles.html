MeshCore::MeshRemoveNeedles Class Reference

#include <Mod/Mesh/App/Core/Degeneration.h>

Detailed Description

The MeshRemoveNeedles class tries to fix degenerations by removing needles. Needles are triangles where its longest edge is much longer than its shortest edge. https://graphics.uni-bielefeld.de/publications/vmv01.pdf
