Gui::GUIConsole Class Reference

#include <Gui/GuiConsole.h>

Detailed Description

The console window class This class opens a console window when instantiated and redirects the stdio streams to it as long it exists. This is for Windows only! After instantiation it automatically registers itself at the FCConsole class and gets all the FCConsoleObserver messages. The class must not used directly! Only the FCConsole class is allowed!
