Gui::UserNavigationStyle Class Referenceabstract

#include <Gui/NavigationStyle.h>

Detailed Description

Sub-classes of this class appear in the preference dialog where users can choose their favorite navigation style. All other classes that inherit directly from NavigationStyle do not appear in the above dialog. This mechanism is useful to implement special navigation styles which are only needed for certain purposes. Thus, it should not be possible to be choosable by the user
